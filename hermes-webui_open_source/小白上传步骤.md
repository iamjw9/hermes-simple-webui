## 📋 上传步骤 (按顺序操作)

### 👉 第一步：打开仓库页面
1. 查看你刚才创建的仓库链接
   - 示例: `https://github.com/[你的用户名]/hermes-webui`
   - 比如我的是: `https://github.com/Jack/hermes-webui`

2. **点击左上角 "Code" 按钮** (或者直接点仓库链接)

3. **选择 "HTTPS" 标签** (确保能看到这个图标: 🔒)

4. **点击 "Copy" 按钮** (复制地址)
   - 这会复制类似: `https://github.com/Jack/hermes-webui.git`

---

### 👉 第二步：打开命令行 (PowerShell 或 CMD)

1. **打开 PowerShell**:
   - 按 `Win + R`
   - 输入 `powershell` 然后回车
   - 或者开始菜单 → 搜索 "PowerShell" → 打开

2. **切换到文件目录**:
   - 复制下面这行命令，粘贴到 PowerShell 回车:
   ```bash
   cd "D:\4. Web\HYCOforce web\0_workstation\2_Kilo\hermes_webui"
   ```
   - 如果提示错误，就用这个确认一下有没有这个文件夹:
   ```bash
   ls
   ```

---

### 👉 第三步：初始化 Git 并上传

在 PowerShell 中依次复制粘贴执行下面这一长串命令 (全部一起执行):

**复制整个代码块**，然后粘贴到 PowerShell 回车:

```bash
cd "D:\4. Web\HYCOforce web\0_workstation\2_Kilo\hermes_webui"

# 第一步：初始化 Git (这是第一次用)
git init

# 第二步：设置你的名字和邮箱 (下一步会要求你输入)
git config user.name "Jack"
git config user.email "jack@example.com"

# 第三步：添加文件到 Git (把开源文件夹里的所有文件上传)
git add hermes-webui_open_source/*.html hermes-webui_open_source/*.py hermes-webui_open_source/*.md hermes-webui_open_source/*.license hermes-webui_open_source/*.nginx

# 第四步：提交更改 (表示同意上传)
git commit -m "Release: Hermes WebUI v20260603 - 双服务器切换 + PTY 终端"

# 第五步：连接到 GitHub (把 [替换成你自己 GitHub 用户名])
git remote add origin https://github.com/[你的GitHub用户名]/hermes-webui.git

# 第六步：推送到 GitHub
git push -u origin main
```

---

### 👉 第四步：输入 GitHub 密码 (重要!)

执行 `git push` 后，会提示输入密码，此时你需要:

1. **打开 GitHub**，登录账号
2. **点击右上角头像**，选择 "Settings"
3. **左侧菜单找到 "Developer settings"** (开发者设置)
4. **点击 "Personal access tokens" (个人访问令牌) → "Tokens (classic)"**
5. **点击 "Generate new token (classic)"**
6. **设置:**
   - Note (备注): `GitHub Upload 2026` (随便写)
   - Expiration (过期时间): 一年或无限制 (无限制也可以)
   - **勾选权限** (全部勾上):
     - ✅ `repo` (全部勾选后自动全部被勾上)
7. **点击 "Generate token"**
8. **复制生成的密码** (只有一次显示机会，记得马上复制!)
   - 密码是 `ghp_非常长的字母数字组合`

9. **回到 PowerShell，把密码粘贴进去**

10. **按回车**，等待上传完成 (可能需要 1-3 分钟)

## ❌ 如果遇到问题

### 问题 1: 提示 "fatal: not a git repository"
→ 文件尾部还没加换行符，在文件夹空白处右键 → New File → 保存所有文件，然后再重新执行 `git add`

### 问题 2: 提示 "git: command not found"
→ Windows 没装 Git，快速安装:
1. 下载: https://git-scm.com/download/win
2. 全部选 Next (一路 Next)
3. 安装好后重启 PowerShell

### 问题 3: 推送时提示 "User is not a collaborator"
→ 确认你创建的仓库是不是和你 GitHub 账户关联的

### 问题 4: 密码不对
→ 重新生成一个新的 token (只能用一次，旧的要删掉)
